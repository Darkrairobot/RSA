# RSA - Software Educacional de Criptografia

Projeto de desenvolvimento de um software educacional baseado no algoritmo de criptografia RSA, com o objetivo de demonstrar, de forma prática e didática, os conceitos fundamentais da criptografia assimétrica.

Este projeto foi desenvolvido no curso de Análise e Desenvolvimento de Sistemas da FATEC de Ribeirão Preto, integrando conhecimentos de programação e matemática.

## Objetivo

Desenvolver uma aplicação simples que permita ao usuário compreender o funcionamento do algoritmo RSA, incluindo geração de chaves, criptografia e descriptografia de mensagens.

---

---

## Funcionalidades

- Validação de números primos  
- Geração de chave pública e privada  
- Criptografia de mensagens  
- Descriptografia de mensagens  
- Menu de escolhas via terminal

---

## Conceitos abordados

- Algoritmo RSA  
- Números primos  
- Máximo Divisor Comum (MDC)  
- Inverso modular  
- Conversão de caracteres para valores numéricos e de valores númericos para caracteres 

---

## Tecnologias utilizadas

- **Linguagem:** Python  
- **Bibliotecas:**
  - **sympy** -> validação de números primos  
  - **math** -> operações matemáticas (MDC, etc.)
